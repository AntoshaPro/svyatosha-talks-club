# SvyaTosha: Quantum Empire Builder

AI-powered command center for building and scaling digital empires of spiritual entrepreneurs. This high-vibration dashboard leverages the power of artificial intelligence (Google Gemini API) to automate and amplify all aspects of personal branding and content creation.

## Features

### Strategic Empire Management ("Empire" Tab)
- Goal tracking dashboard with progress visualization
- Growth projection timeline showing exponential growth
- AI Oracle providing step-by-step strategy for scaling

### Viral Content Laboratory ("Viral Workshop" Tab)
- Personality Calibration: AI analyzes your digital DNA from social media links
- Empathetic Resonance: Deep analysis of audience pain points and desires
- Trend Scanner and Spy: Finds viral trends and competitor analysis
- Neuro-Scout: Searches for free AI video generation tools
- Masterpiece Generation: Creates viral content in various formats

### Mindset Work ("Money Flow" Tab)
- Financial Block Diagnosis: Identify money blocks and fears
- Block Dissolution: AI-powered NLP techniques to transform limiting beliefs
- Universe Assignment: Concrete actions to reinforce new empowered state

## Getting Started

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Set up your environment:
```bash
# Create a .env file with your Google API key
echo "GOOGLE_API_KEY=your_key_here" > .env
```

3. Run the application:
```bash
streamlit run src/main.py
```

Or use the run script:
```bash
python run_app.py
```

## Usage

The application provides four main tabs for different aspects of your digital empire building:

- **Empire Dashboard**: Track goals and get AI-generated strategic advice
- **Viral Workshop**: Create content, analyze trends, and find viral opportunities
