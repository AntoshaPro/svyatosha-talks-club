# Svyatosha Talks Club

AI-powered voice conversation platform for generating and managing dialogue audio content.

## Features

- Voice cloning and synthesis
- Dialogue generation from scripts
- Audio processing and editing
- Integration with social media

## Project Structure

```
.
├── src/              # Source code
├── models/           # ML models
├── data/             # Sample data
└── scripts/          # Utility scripts
```

## Getting Started

```bash
pip install -r requirements.txt
python main.py
```
